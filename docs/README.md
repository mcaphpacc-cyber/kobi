# KOBI

**KOBI (Knowledge of Body Intelligence)** is a lightweight PHP 8.2+ medical knowledge platform built using a custom MVC framework.

The project is designed to provide structured, multilingual medical information using a normalized MariaDB database while remaining fast, maintainable, and framework-independent.

---

## Current Version

**v0.1.0 (Development)**

---

## Technology Stack

- PHP 8.2+
- MariaDB / MySQL
- Composer
- Bootstrap 5.3
- PSR-12 Coding Standard
- Repository Pattern
- MVC Architecture

---

## Project Goals

- Lightweight custom PHP framework
- Fast page rendering
- Clean architecture
- Bilingual support (English & Hindi)
- SEO-friendly URLs
- Medical knowledge database
- Modular design
- Future REST API support
